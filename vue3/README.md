# release-platform

业务发版平台。前身是 lovelive 的 release 模块，用于业务发版。

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
